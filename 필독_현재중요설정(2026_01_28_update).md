# 현재 gabia서버 nginx는 다음과 같이 되어있다. 이렇게 안하면 코드 아무리 수정해도 반영 안된다.

> [!CAUTION]
> **이 nginx 설정은 절대 함부로 수정하지 마라.**
> - `/api/` 프록시 블록을 절대 삭제하면 안 된다. 삭제하면 프론트엔드에서 재고/문서/가격 전부 못 불러온다.
> - `$` 기호는 반드시 `\$`로 이스케이프해야 한다 (bash heredoc 안에서 쉘 변수로 해석되기 때문).
> - 이 설정을 변경할 때는 아래 전체 명령어를 복사해서 그대로 실행해야 한다.

```bash
sudo bash -c 'cat > /etc/nginx/nginx.conf <<EOF
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
pid /run/nginx.pid;

events {
    worker_connections 1024;
}

http {
    include       /etc/nginx/mime.types;
    default_type  application/octet-stream;

    server {
        listen       80;
        server_name  _;
        root         /home/rocky/sammirack-app;
        index        index.html;

        # ✅ index.html은 절대 캐시 금지 (배포 즉시 반영)
        location = /index.html {
            add_header Cache-Control "no-store, no-cache, must-revalidate, proxy-revalidate, max-age=0";
            add_header Pragma "no-cache";
            add_header Expires "0";
            try_files \$uri /index.html;
        }

        # ✅ JSON/CSV 데이터 파일도 캐시 금지
        location ~* \.(json|csv)\$ {
            add_header Cache-Control "no-store, no-cache, must-revalidate, max-age=0";
            add_header Pragma "no-cache";
            try_files \$uri =404;
        }

        # ✅ 해시가 붙은 빌드 산출물은 1년 캐시 (파일명이 바뀌면 자동 갱신)
        location /assets/ {
            add_header Cache-Control "public, max-age=31536000, immutable";
            try_files \$uri =404;
        }

        # ✅ API 프록시 (백엔드 3001 포트로 전달) [절대 삭제 금지!!!]
        # 이 블록이 없으면 프론트엔드에서 재고/문서/가격 데이터를 전혀 불러올 수 없다!
        location /api/ {
            proxy_pass http://127.0.0.1:3001/api/;
            proxy_http_version 1.1;
            proxy_set_header Host \$host;
            proxy_set_header X-Real-IP \$remote_addr;
            proxy_set_header X-Forwarded-For \$proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto \$scheme;
        }

        # 나머지 요청 (SPA 라우팅)
        location / {
            try_files \$uri \$uri/ /index.html;
        }
    }
}
EOF'

sudo systemctl restart nginx
```

## nginx 설정 변경 시 체크리스트

1. ✅ `/api/` 프록시 블록이 있는가? → **없으면 프론트엔드 전체 작동 불가**
2. ✅ `proxy_pass http://127.0.0.1:3001/api/;` 가 정확한가?
3. ✅ 모든 `$` 가 `\$`로 이스케이프 되어있는가?
4. ✅ `index.html`, `json/csv`, `assets` 캐시 설정이 있는가?
5. ✅ SPA 라우팅 `try_files` 가 마지막에 있는가?
